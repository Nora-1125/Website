Web Development

Basics and First Steps

One of the first topics we discussed was web development. We were shown the basics of HTML, a markup language used to structure content on a webpage, and we talked about CSS, which controls the visual style and layout. We also covered JavaScript, the programming language that adds interactivity and dynamic behavior. In this context, we also learned that a markup language is a system for annotating text so that a computer can interpret its structure and meaning.

To see how far I have come: take a look around this website. It was created around the template 'Solid State' from HTML5 UP.
